# React Calculator (Vite)

A responsive calculator web app built with React. Supports chained operations and clean output formatting.

## Features
- Basic operations: +, −, ×, ÷
- Chained calculations
- Clear / reset
- Output formatting (rounding + scientific notation for large values)
- Responsive UI inspired by mobile calculator layout

## Tech Stack
- React
- JavaScript
- Vite
- HTML/CSS

## Getting Started
### Install
```bash
npm install
npm run dev
